ALL TAGS & ATTRIBUTES IN HTML CSS

TAGS Description Syntax
!DOCTYPE html Declares the document type and HTML version. <!DOCTYPE html>
Abbreviation Defines an abbreviation or acronym with a tooltip. <abbr title="">..</abbr>
acronym Defines an acronym with extra semantic meaning. (Deprecated) <acronym title="">...</acronym>
anchor Creates a hyperlink to another page or location. <a href="">...</a>
head Contains metadata and links for the document. <head>...</head>
body Defines the main content of the HTML page. <body>Contents...<body>
html The root element of an HTML document. <html>...</html>
title Specifies the title of the HTML document. <title>Page Title</title>
p Defines a paragraph of text. <p>Paragraph</p>
h1 - h6 Defines HTML headings from largest (h1) to smallest (h6). <h1>Heading</h1>
img Embeds an image in the HTML page. <img src="" alt="">
br Inserts a single line break. <br>
hr Creates a horizontal line or thematic break. <hr>
div Defines a division or section in an HTML document. <div>...</div>
span Defines an inline container for text or elements. <span>...</span>
link Defines the relationship between a document and an external resource. <link rel="stylesheet" href="">
