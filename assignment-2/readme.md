Assignment 2 – Personal Portfolio Page (HTML + CSS)
 Description
This assignment is a Personal Portfolio Web Page created using HTML and CSS.
The page contains multiple sections such as Home, Experience, Skills, Projects, Achievements, and Footer.
The portfolio displays personal information, projects, achievements, and technical skills in a clean and responsive layout.
 Features
●Navigation bar with section links
●Hero / Home section with profile and introduction
●Experience section
●Skills section with tags
●Projects grid layout
●Achievements section with images
●Footer with contact links
●Smooth scrolling effect
●Sticky navigation bar
●Responsive layout using CSS
Technologies Used
●HTML5
●CSS3
●Flexbox
●Grid
●Internal & External Styling
 Files Included
●index.html
●style.css
●images folder (profile, achievements, logos)
How to Run
1.Download or clone the repository
2.Open the folder
3.Open index.html in browser
 Objective
To design a Personal Portfolio Website using HTML and CSS and understand page layout, styling, and navigation.
